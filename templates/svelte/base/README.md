## Updating

To update this template you should:

1. Run `npm create vite@latest` and follow the prompts for a Svelte projects
2. Copy the contents over the top of this folder
3. Use a diff tool to work out what to keep and what to discard.

Generally the `src/` folder and `index.html` file and modifications to
`package.json` should be kept.
